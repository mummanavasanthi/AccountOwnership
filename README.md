# Ethereum Account & Ownership Example

This project demonstrates the deployment of an Ethereum Account & Ownership Example smart contract on SecureChain (SCAI) Mainnet using Solidity, Remix IDE, and MetaMask wallet.

## Technologies Used

- Solidity
- Remix IDE
- MetaMask
- SecureChain Mainnet (SCAI)

## Features

- Store contract owner address
- Verify ownership using Ethereum account
- Demonstrates constructor usage
- Smart contract deployment on SecureChain Mainnet

## Deployment Process

1. Installed MetaMask wallet
2. Added SecureChain Mainnet network
3. Received SCAI faucet tokens
4. Created Account Ownership smart contract in Remix IDE
5. Compiled Solidity contract
6. Connected MetaMask with Remix
7. Deployed contract successfully on SecureChain Mainnet

## Outcome

Successfully deployed the Ethereum Account & Ownership Example smart contract on SecureChain Mainnet using Solidity, Remix IDE, and MetaMask wallet.
